#Gym Management System
#Overview

The Gym Management System is a Flutter-based mobile application designed to help gym owners, trainers, and clients efficiently manage gym operations. The app uses SQLite for local data storage, allowing smooth and offline management of memberships, clients, payments, and analytics.

#Features

Client Management: Add, edit, delete, and view client profiles.

Membership Management: Track active, expired, and upcoming memberships.

Payment Tracking: Record and monitor membership payments.

Search Functionality: Quickly search clients by name or ID.

Dashboard & Analytics: Visualize gym performance with charts and stats.

Offline Support: All data is stored locally using SQLite, so the app works without internet.

#Tech Stack

Frontend: Flutter

Backend / Local Database: SQLite (sqflite package)

State Management: Provider 

Charts & Analytics: Flutter chart libraries

#Installation

Clone the repository:
git clone origin https://github.com/talbianis/Gym_manager.git

Navigate to the project directory:
cd Gym_manager

Install dependencies:
flutter run

#Usage

Manager: Can add clients, manage memberships, track payments, search for clients, and view analytics.

Client: Can view their profile, membership status, and payment history.

#Database

The app uses SQLite to store data locally on the device.

Tables include: clients, memberships, payments, etc.

Supports CRUD operations for all tables.
